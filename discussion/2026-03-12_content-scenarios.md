# 2026-03-12: Card News Content Scenarios

## 카드뉴스의 3가지 콘텐츠 접근법

### Scenario A: 프롬프트 중심 (Prompt-First)
실제 프롬프트를 크게 보여주고 → 결과물 캡처 → `/docx` 같은 슬래시 명령어 소개 → 다른 숨겨진 기능 티저.
연구자가 "나도 이거 치면 되겠다"고 느끼게.

**적합한 시리즈**: Series 1 (Researcher Toolkit) 중 기능별 소개
**슬라이드 구조**: 기능 소개 → 프롬프트 → 결과 → 활용 팁

### Scenario B: 기능 카탈로그 (Feature Catalog)
Claude Code의 연구자용 숨겨진 기능들을 하나씩 소개.
/docx, /pptx, Custom Skills, CLAUDE.md, /memory 등.
각 기능별 1슬라이드.

**적합한 시리즈**: Series 2 (Research Intuition → Claude)
**슬라이드 구조**: 기능명 → 한 줄 설명 → 프롬프트 예시 → 결과

### Scenario C: 시나리오 스토리 ★ SELECTED for #1
한 대학원생이 제안서 마감에 쫓기는 스토리 → Claude Code 발견 → 프롬프트 입력 → 결과 확인 → 감탄.
감정 흐름 중심.

**적합한 시리즈**: Series 1 첫 번째 카드뉴스 (최초 접점)
**슬라이드 구조**:

| # | 모드 | 내용 | 감정 |
|---|------|------|------|
| 1 | ACCENT | 훅: "논문 마감 3시간 전" | 위기감 |
| 2 | WARM | 공감: 서식 지옥의 현실 | 좌절 |

## 핵심 원칙
- 연구자는 APA 7th가 뭔지 이미 안다 → 설명하지 마라
- 연구자는 Claude Code가 뭔지 모른다 → 구체적으로 보여줘라
- 프롬프트는 복사해서 바로 쓸 수 있을 만큼 구체적이어야 한다
- 감정 흐름: 위기 → 좌절 → 호기심 → 기대 → 놀라움 → 감탄 → 확신 → 행동
